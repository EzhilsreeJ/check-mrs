# project-overview-this

## Project Overview
This project is a React-based movie booking system. It allows users to browse movies, view details, select showtimes, choose seats, and complete a booking. The frontend is built with React, utilizing functional components and hooks. Styling is handled with CSS modules for scoped styles.
